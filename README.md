# babel-lib-rollup-starter

Boilerplate for creating JavaScript libraries using [Babel](https://github.com/babel/babel) and [Rollup](https://github.com/rollup/rollup).

## Install
```
$ git clone https://github.com/Banttu/babel-lib-rollup-starter.git
$ cd babel-lib-rollup-starter
$ npm install
```

## Build
```
$ npm run build
```

## Watch for changes
```
$ npm run dev
```
